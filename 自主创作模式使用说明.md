# ScriptAgent 自主创作模式使用说明

## 🎯 功能概述
增强后的ScriptAgent新增了**自主创作模式**作为默认优先模式，Agent可以完全凭借内置的专业带货视频编剧经验，仅根据商品信息自主生成高质量的短视频剧本，不需要调用任何外部工具或查询参考数据，生成速度更快，灵活性更高。

## ✨ 核心优势
1. **生成速度快**：不需要调用外部工具查询模板、爆款视频等，大幅缩短生成时间
2. **灵活性高**：不受模板或参考案例的限制，可以根据商品特点定制化创作
3. **稳定性强**：不依赖外部服务，不会因为工具调用失败而影响生成
4. **专业性强**：内置了成熟的短视频创作方法论和爆款规律

## 🧠 内置创作框架
自主创作模式内置了专业的短视频创作方法论，确保生成的剧本符合平台规律和高转化率要求：

### 黄金结构（AIDA模型）
```
开场Hook → 兴趣激发 → 欲望唤醒 → 行动号召
```

### 场景标准配置
| 场景序号 | 类型 | 时长 | 作用 |
|---------|------|------|------|
| 1 | hook | 3-5秒 | 黄金3秒抓注意力，用痛点、反问、惊喜瞬间留住用户 |
| 2 | selling_point | 4-6秒 | 展示核心卖点，讲清楚产品解决什么问题 |
| 3 | detail/social_proof | 3-5秒 | 细节展示/口碑背书，建立用户信任 |
| 4 | price/cta | 2-4秒 | 价格优惠+行动号召，引导用户下单 |

### 内容优化原则
- **口语化文案**：避免书面语，像真人带货一样自然有感染力
- **画面描述详细**：image_prompt包含主体、动作、背景、光线、色调、构图，可直接用于AI生成
- **短字幕强化**：overlay.text不超过10个字，突出核心信息
- **节奏紧凑**：总时长12-20秒，符合短视频完播率规律

## 🚀 使用方法

### 1. API接口调用
在调用生成接口时，指定`creation_mode: "independent"`即可：
```javascript
// 前端调用示例
fetch('/script/v1/generate', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        project_id: 123,
        creation_mode: "independent",  // 明确指定自主创作模式
        force: false
    })
})
```

### 2. 代码中直接调用
```python
from backend.v1.app.agent.implementations.script_agent import ScriptAgent

# 初始化Agent
agent = ScriptAgent()

# 商品信息
product_info = {
    "商品标题": "无线蓝牙耳机",
    "商品描述": "蓝牙5.3，HiFi音质，24小时续航，IPX7防水",
    "视频风格": "时尚活力",
    "目标受众": "18-35岁年轻人",
    "重点强调": "音质好、续航长、防水"
}

# 调用自主创作模式
script = await agent.generate_script(
    project_info=product_info,
    target_duration=15,
    creation_mode="independent"  # 指定自主创作模式
)
```

## 📝 输入参数说明
自主创作模式只需要提供基础的商品信息即可生成高质量剧本，推荐提供以下信息（越详细生成效果越好）：

| 字段 | 说明 | 是否必填 |
|------|------|----------|
| 商品标题 | 商品的名称 | ✅ 是 |
| 商品描述 | 商品的核心功能和特点 | ✅ 是 |
| 用户创作意图 | 用户的特殊要求，比如突出什么特点 | ⭕ 推荐 |
| 视频风格 | 视频的整体风格，如时尚活力、清新自然等 | ⭕ 推荐 |
| 目标受众 | 目标用户群体，如18-25岁女大学生 | ⭕ 推荐 |
| 重点强调 | 需要重点突出的卖点列表 | ⭕ 推荐 |
| 需要避免 | 创作中需要避免的内容 | ⭕ 可选 |
| 价格/原价 | 商品价格信息，用于生成价格场景 | ⭕ 推荐 |
| 商品分类 | 商品所属类别，如数码配件、美妆护肤 | ⭕ 可选 |
| 商品详情 | 更详细的商品参数信息 | ⭕ 可选 |

## 🔄 模式对比
| 创作模式 | 适用场景 | 优点 | 缺点 |
|---------|----------|------|------|
| **自主创作（默认）** | 大多数场景，需要快速生成 | 速度快、稳定、灵活 | 没有参考现成爆款案例 |
| 爆款融合 | 需要参考同类爆款视频 | 参考成功案例，转化率有保障 | 生成速度慢，依赖视频库 |
| 模板生成 | 需要保持特定风格/结构 | 风格统一，质量稳定 | 灵活性差，受模板限制 |
| 策略因子 | 需要特定策略指导创作 | 符合策略规则，创新性强 | 依赖策略库 |

## 💡 最佳实践
1. **常规场景优先使用自主创作模式**：速度快，效果稳定，满足大多数需求
2. **提供尽量详细的商品信息**：信息越丰富，生成的剧本越贴合产品特点
3. **明确重点强调内容**：告诉Agent需要重点突出哪些卖点，生成更精准
4. **生成后可通过修改接口微调**：如果对生成结果不满意，可以通过revise接口进行多轮修改

## 📌 注意事项
- 自主创作模式是默认模式，即使不指定creation_mode也会优先使用
- 只有当用户明确要求参考爆款、使用模板或策略时，才需要切换到其他模式
- Agent会根据商品特点自动调整场景数量和结构，不需要手动指定
